# 🌿 KeenKeeper — Keep Your Friendships Alive

## 📌 Project Description

KeenKeeper is a modern friendship tracking web application built with Next.js.  
It helps users stay connected with their friends by logging interactions like calls, texts, and video chats.

Every interaction is recorded in a timeline and visualized in a stats dashboard using a dynamic pie chart.  
The goal of this project is to make maintaining friendships more meaningful, organized, and visually engaging.

---

## ✨ Features

- 👥 **Friend List & Profile System**  
  Browse friends on the home page and view detailed information by clicking on any friend card.

- 📞 **Interaction System (Call / Text / Video)**  
  Perform actions like call, text, or video chat. Each action triggers a toast notification and logs the activity.

- ⏱️ **Timeline History**  
  All interactions are stored in a timeline for easy tracking of friendship activities.

- 📊 **Statistics Dashboard**  
  Visual representation of user interactions (calls, texts, video calls) using a dynamic pie chart.
